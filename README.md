aobench_haxe
============

aobench in haxe

haXe
------
### requirement ###
+ haXe (http://haxe.org/)
+ html5 (https://github.com/tong/hx.html5)

    `$ haxelib install html5`

### build ###
    $ cd haxe
    $ haxe comp.hxml
